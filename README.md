# Penggunaan Manual

## Install Plugin
```php
composer install
```
```php
php artisan serve
```

## Generate dummy data
```php
php artisan migrate --seed
```
Seandainya seed database tidak jalan. maka gunakan perintah dibawah
```php
php artisan db:seed --class=AccessLogSeeder
```
